2 Generate & copy scraper
Click Generate, then copy and paste the scraper into the Geometrack console on any case detail page. It will automatically open and fill the grievance form.

How to use — once per case page

  1. Open a Geometrack case detail page (e.g. MR2600288)
  2. Click the Related Mill Group tab to load mill data
  3. Press F12 → click the Console tab
  4. Type allow pasting and press Enter (first time only per browser session)
  5. Click Copy Scraper Code above → switch to Geometrack → press Ctrl+V → Enter
  6. The grievance form opens automatically with mills pre-filled ✓
